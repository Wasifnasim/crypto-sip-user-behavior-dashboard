# Crypto SIP User Behavior & Retention Dashboard

##  Project Overview
This project is an interactive **Tableau dashboard** built to analyze user behavior in a **Crypto SIP (Systematic Investment Plan) platform** using a simulated dataset of 500 users.  

The dashboard focuses on understanding **user retention, investment behavior, and growth trends** to provide business insights that can help fintech platforms improve customer engagement.

---

##  Business Objective
The main goal of this project is to:

- Analyze user behavior in a Crypto SIP platform
- Identify retention and drop-off patterns
- Understand investment trends across different SIP frequencies
- Generate insights that support business growth and user engagement

---

##  Dashboard Features

### KPI Cards
- Total Users
- Active Users
- Stopped Users
- Retention Rate

### Visual Analysis
- Active vs Dropped Users (Pie Chart)
- SIP Frequency vs Users (Bar Chart)
- User Growth Trend Over Time (Line Chart)
- Average SIP Amount by Frequency
- Average Investments by User Status

### Interactive Filters
- Frequency
- Age
- SIP Amount

---

##  Key Insights
- User retention analysis
- Active vs dropped users comparison
- SIP frequency behavior trends
- Average SIP investment analysis
- User growth trends over time
- Investment behavior by user status

---

## Tools & Technologies Used
- **Tableau** (Dashboard & Visualization)
- **Python** (Dataset generation)
- **Pandas** (Data handling)
- **CSV Dataset**

---

##  Dataset Information
A simulated dataset of **500 users** was created to model real-world Crypto SIP user behavior.

Dataset includes:

- User ID
- Age
- SIP Amount
- Frequency (Daily / Weekly / Monthly)
- Start Date
- Last Investment Date
- Total Investments
- User Status (Active / Dropped)

This dashboard demonstrates how fintech companies can use **data-driven insights** to:

- Improve user retention

- Optimize investment engagement
